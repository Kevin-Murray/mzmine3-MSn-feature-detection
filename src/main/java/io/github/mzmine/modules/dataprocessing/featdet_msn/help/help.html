MSn feature list builder

Description

The MSn feature list builder modules scans the raw data looking for MSn scans and builds a chromatographic feature for each MS1 precursor ion with a corresponding MSn scan.

Using the MS Level parameter input in the Scan Selection field, the specified MSn scans are subset and the precursor MZ is collected for each scan. The precursor MZ value is obtained from the corresponding MS2 scan. If the MS Level is greater than 2, the original isolating MZ the MSn scans corresponding MS2 is used as the precursor MZ. The retention time of the scan and precursor MZ are used to build a feature chromatogram for each detected MSn scan.

A feature list of precursor signals is constructed using the specified mass and retention time tolerances. For each detected MSn-containing precursor signal, a feature chromatogram in the retention time range is built using all MS1 basepeaks within the mass range.

If more than one peak is detected in the range or more precise peak integration is required, the feature list may be deconvoluted via the Chromatogram resolving tools. If more than one MSn scan triggered on the same precursor ion, duplicate feature chromatograms may be built. Use the Feature list filtering > Duplicate feature filter tool to remove redundant results from the generated feature list.

Note - MSn files may not be correctly detected if Raw data files were converted using older versions of MSConvert or ThermoRawFileParserer. If no MSn scans are detected in the dataset, re-convert the using the latest version of these tools to avoid complications.

Method parameters

Scan Selection
Scan selection window for MSn feature detection. MS Level must be set in Scan Selection field.
m/z Tolerance
Precursor ion (MS1) mass tolerance for feature building. All MS1 base peaks within this mass range will be use to build a targeted chromatogram feature.
Retention time tolerance
Precursor ion (MS1) retention time tolerance for feature building. All MS1 base peaks with this retention time range will be used to build a targeted chromatogram feature.
